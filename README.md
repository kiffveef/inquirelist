# 問合せメール

## メール送信

### 1. メールテンプレートを作成
![メールテンプレート](img/mailtemplate_01.png)

設定にしたがってメールを作成します。  
以下の項目の入力は必須です。
1. 差出人
2. 件名
3. 本文  
本文については、置換文字列 `{company}` が使用できます。  
使用することでメール宛先リストにある**企業名**に置換されます。

### 2. メール送信
メールテンプレートの必須項目が入力され、かつメール宛先リストに有効なデータがある場合メール送信します。  
右上 `メニュー` より `メール送信` を選択することでメール送信を実行できます。
